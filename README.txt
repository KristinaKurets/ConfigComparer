Hello! This is a comparer for XML config files.
If you want to find keys, that have different values in different config files, this program will be useful for you :) 
Lets's start!

1. Open the ConfigComparer.sln file.
2. Then find and open app.config file and change keys 'CloudProjectPath'and 'LoyaltyProjectPath' with your projects paths.
3. Also you can change search file and path for your logging file in keys 'SearchFile' and 'LoggerPath', respectively.
4. Now, after all your changes, run the project.
5. Result of your comparing you can see in the logging file.

That's all!
Enjoy :)
